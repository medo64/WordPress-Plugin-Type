Allows better <pre> formatting via simple short code.

== Description ==

Whenever [pre][/pre] is encountered in post, it will clean it up
in order to provide better <pre> formatting.

== Installation ==

1. Extract plugin zip file into `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
